# Content Elevated Asset Library

Use this folder to keep product visuals out of Payhip-only storage.

## Folders

- `product-covers/`: final product cover images used for Payhip, the website, and launch marketing.
- `product-mockups/`: alternate mockups, lifestyle product graphics, and bundle presentations.
- `payhip-screenshots/`: screenshots of live Payhip listings for reference before/after updates.

## Naming

Use product slugs so assets are easy to connect to product data:

- `hair-stylists-cover.png`
- `hair-stylists-mockup-01.png`
- `hair-stylists-payhip-before.png`
