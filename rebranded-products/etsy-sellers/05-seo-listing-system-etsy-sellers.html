✓ ORGANISING YOUR KEYWORDS:

✓ Create a simple spreadsheet: column 1 = keyword, column 2 = search volume

estimate (high/medium/low), column 3 = competition (high/medium/low). Prioritise medium competition + high search volume first.

SECTION 5: STAR SELLER ACHIEVEMENT

CHECKLIST

✓ STAR SELLER REQUIREMENTS (Etsy reviews these monthly):

✓ Message response rate: 95%+ of messages replied to within 24 hours✓

✓ Dispatch rate: 95%+ of orders dispatched on or before the dispatch date✓

✓ 5-star reviews: 4.8+ average with a minimum of 5 reviews in the period✓

✓

✓ SYSTEMS TO ACHIEVE STAR SELLER:

✓

✓ Message response (24hrs target):

✓ - Set up Etsy notifications on your phone — respond to new messages immediately

✓ - Save canned responses for your most common questions using the prompts in

Doc 3

✓ - If you cannot respond fully within 24hrs, send a brief 'received your message, will

respond fully by [time]' acknowledgement

✓

✓ Dispatch rate (on-time target):

✓ - Only set processing times you can genuinely meet — do not over-promise

✓ - Build buffer time into your processing window during busy seasons

✓ - If you know you will be delayed, message the customer proactively before they

contact you

✓

✓ Review generation (4.8+ target):

✓ - Use the post-delivery follow-up message template from Doc 3 for every order

✓ - Include a packaging insert / thank you card with every order asking for a review

✓ - Respond to every review (good and bad) within 24 hours — this signals

professionalism

✓ - Never ask for or incentivise positive reviews — this violates Etsy's policies
