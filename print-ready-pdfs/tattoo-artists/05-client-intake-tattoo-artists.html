SECTION 2: CONSULTATION PROCESS GUIDE

How to Run a Productive Consultation

✓ BEFORE THE CONSULTATION:

✓ Review the intake form completely — come with specific questions

✓ Look up any reference styles or artists the client mentioned

✓ Have your portfolio ready to show relevant examples

✓ Prepare rough size/pricing estimates for the type of work

✓

✓ THE CONSULTATION (20-30 minutes):

✓ 1. Start with open questions: 'Tell me more about the idea — what drew you to it?'

✓ 2. Show relevant portfolio examples: 'This is similar to what you're describing —

does this feel right?'

✓ 3. Discuss placement together: 'For this type of design, these placements work

particularly well because...'

✓ 4. Talk size honestly: 'For the level of detail you want, this would work better at [X

size] — here's why...'

✓ 5. Cover timeline: 'A piece like this would take approximately [X] hours. We could do

it in one session or split it.'
